# Tennis Coaching App

### This application is built in Vue 3 using Vuex, Jest for TDD/Unit Testing and Tailwind for layout design

## Project setup
```
npm install
```

### Run development environment server
```
npm run serve
```

### Open on browser
```
http://localhost:8080
```

### Run tests
```
npm run test:unit
```
